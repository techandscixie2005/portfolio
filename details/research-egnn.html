RTP Property Prediction with Equivariant Graph Neural Networks

Supervisor: Prof. Jun Jiang
Fall 2025 - Spring 2026
Graduate Project

Overview

This project focuses on predicting room-temperature phosphorescence (RTP) properties of organic molecules using state-of-the-art equivariant graph neural networks (EGNNs). RTP is a fascinating photophysical phenomenon with applications in organic electronics, bioimaging, and information encryption.

The challenge lies in accurately predicting key photophysical properties such as singlet oxygen quantum yield (ΦSO) and phosphorescence energy gap (ΔEST), which are computationally expensive to obtain through traditional quantum mechanical methods.

Dataset

We compiled a dataset of 6,000 RTP-active organic molecules with DFT-calculated photophysical properties:

  • Singlet oxygen quantum yield (ΦSO)
  • Phosphorescence energy gap (ΔEST)
  • Radiative and non-radiative decay rates
  • Molecular structures with optimized geometries
  • Electronic properties (HOMO/LUMO energies, band gaps)

Model Architecture

We implemented an 8-layer EGNN_Sparse_Network with SE(3)-equivariant message passing, which preserves rotational symmetry throughout the network:

  • Node Features (21-dim): Atom type, hybridization state, aromaticity, valence electrons, etc.
  • Edge Features (6-dim): Bond type, bond length, distance in molecular graph
  • Message Passing: SE(3)-equivariant transformations on geometric tensors
  • Output: Scalar properties predicted through invariant aggregation

Training Strategy

  • Geometric Data Augmentation: Random rotations and translations of molecular coordinates
  • Regularization: Layer-wise Dropout (0.1-0.3) to prevent overfitting
  • Learning Rate Schedule: ReduceLROnPlateau with patience=10
  • Loss Function: Mean Squared Error (MSE) with optional MAE
  • Train/Val/Test Split: 70/15/15 with scaffold splitting

Key Insights

  • SE(3)-equivariance significantly improves prediction accuracy compared to invariant models
  • Geometric data augmentation helps the model generalize to unseen molecular scaffolds
  • The model captures subtle relationships between molecular geometry and electronic properties
  • Interpretability analysis reveals important structural motifs for RTP design
